# House Price Prediction

A machine learning project to predict house prices. The project includes model training using scikit-learn and a simple interactive interface built with Streamlit.

---

## 🛠 Technologies & Libraries

- **Python **  
- **pandas** – for data manipulatio  
- **numpy** – for numerical computations  
- **scikit-learn** – for model training and evaluation  
- **joblib** – to save/load trained models  
- **Streamlit** – for interactive web app  
- Other libraries used for preprocessing and visualization

---

## 💡 Project Overview

1. Load and preprocess the house price dataset  
2. Train a machine learning regression model  
3. Evaluate model performance  
4. Save the trained model using Joblib  
5. Deploy an interactive Streamlit app to make predictions
